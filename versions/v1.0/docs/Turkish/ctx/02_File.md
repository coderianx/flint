# Ctx/02 | `ctx.File()`

```go
package main

import (
    "github.com/grayvort3x/Flint"
)

func main() {
    app := flint.NewServer()

    app.Handle("/", func(ctx *flint.Context) {
        ctx.File("example.html")
    })

    app.Run(":8080")
}
```
