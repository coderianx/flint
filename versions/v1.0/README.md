## üöÄ Quick Start

```bash
go get github.com/coderianx/Flint
```

```go
package main

import "github.com/coderianx/Flint"

func main() {
    app := flint.New()

    app.Handle("/", func(c flint.Context) {
        ctx.String("Hello from Flint!")
    })

    app.Listen(":8080")
}
```
